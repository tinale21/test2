## Project 1: The "Hero Faction" Screen (20%) — Due 4/8/26
A visually stunning, interactive single-page web experience where players choose their faction/character. Hosted live on GitHub Pages.

**Vibe:** Interactive poster. Hover states react — typography shifts, unselected options fade.

**Deliverables:**
- Live Hero Screen on GitHub Pages
- Design Intent (Soft PRD — written *before* AI touches code)
- Mermaid diagram (system flow: input → process → output)
- AI Direction Log (3–5 entries)
- Records of Resistance (3 documented AI rejections/revisions)
- Five Questions reflection (README or sketchbook)

**Milestone Schedule:**

## ESF Framework (applies to all projects)
- **Design Intent** — Creative spec written before AI engagement
- **AI Direction Log** — 3–5 entries: what you asked, what AI produced, what you kept/rejected/changed and why
- **Records of Resistance** — 3 documented moments of rejecting/revising AI output
- **Five Questions Reflection** — Self-awareness paragraph before submission
- **Mermaid Diagram** — System flow in the GitHub repo
