## The Vibe
This is an **interactive poster**. It needs to drip with atmosphere.
- Hover over a character → layout reacts
- Typography shifts
- Unselected options fade into the background

**Design Goal:** Masterful typography, CSS Grid layouts, atmospheric hover states.

---

## What You Turn In

Everything ships together in the GitHub repo / GitHub Pages URL.

1. **Live Hero Screen URL** — Single-page interactive experience. Stunning atmosphere. Hosted and accessible.

2. **Design Intent** — Creative spec written *before* AI engagement. Must include:
   - Color palette
   - Typographic hierarchy
   - Hover-state rules
   - Mood/atmosphere

3. **Mermaid Diagram** — System flow in the GitHub repo:
   - What receives input
   - How the system processes it
   - What it outputs

4. **AI Direction Log (3–5 entries)** — For each entry:
   - What you asked AI to do
   - What it produced
   - What you changed / rejected / kept, and why
   - (Can be in README, annotated commits, or sketchbook)

5. **Records of Resistance (3 moments)** — For each:
   - What did AI produce?
   - Why did you reject or revise it?
   - What did you do instead?

6. **Five Questions Reflection** — Short paragraph in README or sketchbook, completed before submission.

## Grading Rubric (100 pts)

| Category | Pts | Meets Expectations | Below Expectations |
|----------|-----|-------------------|-------------------|
| Live Hero Screen | 30 | Stunning atmosphere. Typography, color, hover states unified. Layout responds to interaction. Hosted. | Flat, generic, non-interactive. No hover. Broken or not hosted. |
| Design Intent | 15 | Clear visual rules: palette, type hierarchy, hover behavior, mood. Written before AI coding. Specific enough to evaluate against. | Vague, written after the fact, or missing. |
| Mermaid Diagram | 10 | Accurate system flow: input, processing, output. Matches built project. | Missing, inaccurate, or doesn't reflect actual project. |
| AI Direction Log | 15 | 3–5 entries showing what was asked, produced, and decided. Demonstrates editorial judgment. | Missing, <3 entries, or generic ("I used AI for help"). |
| Records of Resistance | 15 | 3 documented rejections/revisions. Names what AI gave, what you did instead, and why. | Missing or shallow. No evidence of critical evaluation. |
| Five Questions | 10 | Honest, specific. Demonstrates self-awareness about the process. | Missing, perfunctory, or unreflective. |
| Studio Crit Presentation | 5 | Can walk through the project and explain creative/technical decisions. Engages with crit. | Cannot explain choices. Disengaged. |
