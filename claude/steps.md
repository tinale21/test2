# Steps

## Local Development Setup
1. Clone `https://github.com/tinale21/test2`
2. Run `npm install`
3. Run `npm run dev` — app runs at `http://localhost:5173/test2/`

## Build & Deploy
1. Run `npm run build` — output goes to `dist/`
2. Push to `main` — GitHub Actions builds and deploys to GitHub Pages automatically
3. Live at `https://tinale21.github.io/test2/`

## GitHub Pages Setup (one-time)
- Repo Settings → Pages → Source → GitHub Actions

## Asset Locations
- Character PNGs: `public/characters/` (kevin.png, hanna.png, rae.png, daniel.png)
- Character emblems: `public/emblems/` (kevin.png, hanna.png, rae.png, daniel.png)
- Background image: `public/bg.png`
